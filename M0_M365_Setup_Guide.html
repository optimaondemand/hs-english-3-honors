M365 SETUP & FILE ORGANIZATION
Where your work lives, and how to name it
Almost everything you turn in this year is a Microsoft Word document. Set this up once now and every submission will be quick.
💻 STEP 1 · OPEN YOUR M365 TOOLS

Sign in to Microsoft 365 with your Optima account. You will mostly use Word for writing, with PowerPoint and Excel now and then for a feature or a planning grid. Everything you create saves automatically to OneDrive, your cloud storage, so your work is safe and reachable from any device.

📁 STEP 2 · BUILD YOUR FOLDER

In OneDrive, make this folder path so every English III Honors file has one home:

OneDrive
📁 Optima
📁 ELA
📁 ELA11  ← all your English III Honors work goes here
🏷️ STEP 3 · NAME EVERY FILE THE SAME WAY

Use this exact pattern so your teacher can find your work instantly:

LastName_FirstName_AssignmentShortName_ELA11
Example: Rivera_Mia_TwelveLines_ELA11
When an assignment has drafts, add the stage at the very end, after ELA11: _Draft1, _Draft2, _Final.
Example: Rivera_Mia_ElsinoreStage_ELA11_Draft1
📤 STEP 4 · SUBMIT THROUGH CANVAS

Open the assignment in Canvas and choose the OneDrive tab in the submission box. Find your file in ELA11, select it, and submit. Because it is stored in OneDrive, your teacher always sees your most recent saved version. If the OneDrive tab ever gives you trouble, you can also upload the file directly; email your facilitator if you get stuck.

💡 ONE HABIT THAT SAVES YOU

Name the file correctly the moment you create it, before you write a word. It takes five seconds then and prevents lost or misfiled work later.
